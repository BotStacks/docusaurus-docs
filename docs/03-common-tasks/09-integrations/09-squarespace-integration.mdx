# Integrate BotStacks with Squarespace

This guide will walk you through adding the BotStacks assistant widget to your Squarespace website using the custom code feature.

## 1. Copy your BotStacks widget code

Go to the **Activate** tab in your BotStacks dashboard. You'll see the code snippets you need to add to your website.

<img alt="Activate tab with widget code" src="/images/integrations/activate.png" />

You'll need two pieces of code:
- A `<meta>` tag for the Header code section
- Two `<script>` tags for the Footer code section

### Meta tag:

```html
<meta charset="UTF-8">
```

### Script tags:

> **Note:** Be sure to copy your own `apiKey` and `botId` from the Activate tab in your BotStacks dashboard. Do not use the example values below.

```html
<script type="text/javascript" src="https://botstacks.sfo3.cdn.digitaloceanspaces.com/webassistant.js"></script>
<script type="text/javascript">
  window.onload = function() {
    initBotStackAssistant({
      apiKey: "YOUR_API_KEY_HERE",
      botId: "YOUR_BOT_ID_HERE"
    });
  };
</script>
```

## 2. Add the code to your Squarespace site

1. In your Squarespace site, click **Custom Code** in the left sidebar.
2. Click **Code Injection** from the dropdown menu.

<img alt="Squarespace Custom Code navigation highlighted" src="/images/integrations/squarespace.png" />

3. In the **Header** code box, paste the `<meta charset="UTF-8">` tag.
4. In the **Footer** code box, paste both `<script>` tags from above.
5. Click **Save**.

<img alt="Squarespace Code Injection panel with Header and Footer sections" src="/images/integrations/squarespace_code_injection.png" />

## 3. Publish your site

1. Your changes are automatically saved and published.
2. Visit your site and confirm the BotStacks widget appears.
